# smpl. 
### 90+ Browser Tools. Local-First. No Uploads.

**[smpl.codes](https://smpl.codes/)** is a collection of essential web utilities for developers, designers, and everyday users. Unlike traditional online converters, **smpl.** processes everything inside your browser. Your files, passwords, and data never touch a server.

---

## ✨ Features
* **100% Client-Side:** Powered by JavaScript and WebAssembly.
* **Privacy by Design:** No tracking, no analytics, and zero data collection.
* **Offline Ready:** Works without an internet connection once loaded.
* **Lightning Fast:** No upload/download wait times—processing happens at CPU speed.
